Objetivo del proyecto

Kayrox Code (KCIDE) es un entorno de desarrollo integrado (IDE) con identidad propia, potenciado por Mikota Core. Su propósito es brindar estabilidad, rendimiento y compatibilidad con los proyectos de Kayrox, además de crear un espacio donde los desarrolladores puedan trabajar con un lenguaje propio: Kayrox MikCode (.mkay). El objetivo principal es diferenciarse de los IDEs genéricos, ofreciendo una experiencia visual y funcional adaptada a las necesidades de la comunidad Kayrox.

Análisis del código MikCode

El fragmento de código compartido define la estructura básica del lenguaje Kayrox MikCode y muestra cómo se encapsulan metadatos, compatibilidad y entorno:

Arquitectura del sistema

El sistema se compone de varios elementos clave: el núcleo Mikota Core, que gestiona compatibilidad y rendimiento; la interfaz WinForms, que ofrece una UI clásica con menús personalizados; la extensión para VS Code, que proporciona resaltado de sintaxis y configuración; y el lenguaje Kayrox MikCode, un DSL con identidad propia y metadatos de versión.

Características principales

Entre las funciones más destacadas se encuentran el resaltado de sintaxis para archivos .mkay, los temas visuales corporativos, la compatibilidad con Windows y VS Code, el soporte activo con actualizaciones periódicas y la integración con proyectos Kayrox existentes.

Estado actual

Actualmente el proyecto se encuentra en fase beta, con soporte vigente y evolución constante del núcleo Mikota Core. Es compatible con Windows (Visual Studio 2019, WinForms) y Visual Studio Code. La distribución se realiza mediante paquetes .vsix para la extensión.

Próximos pasos

Las metas de desarrollo incluyen la optimización del núcleo Mikota Core, la ampliación de funciones en la extensión de VS Code (autocompletado, snippets, depuración), la documentación detallada de la sintaxis MikCode, pruebas de estabilidad y liberación de versiones candidatas, así como la integración con sistemas de control de versiones y herramientas de colaboración.

Riesgos y retos

El proyecto enfrenta desafíos como errores de instalación (npm/vsce), limitaciones de compatibilidad al estar disponible solo para Windows, dependencia del núcleo Mikota Core que requiere pruebas exhaustivas, y la curva de aprendizaje que implica adaptarse al nuevo lenguaje y entorno.

Conclusión

Kayrox Code representa un paso importante en la consolidación de la identidad tecnológica de Kayrox. Al integrar un núcleo propio (Mikota Core), un lenguaje especializado (.mkay) y soporte en plataformas reconocidas como Visual Studio y VS Code, el proyecto busca posicionarse como una alternativa sólida y diferenciada en el mundo del desarrollo. La documentación aquí presentada servirá como guía para el equipo de desarrollo y como referencia para futuros colaboradores, asegurando que el proyecto mantenga una evolución clara y organizada.
