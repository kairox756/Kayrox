Como Crear Un Archivo HTML en VS Code

Aprende a crear tu primer archivo HTML en el editor de código VS Code.

Muchos de los alumnos del 602 T.V han tenido dificultades para crear un archivo HTML en Visual Studio Code
para eso creamos esta guia rapida y sencilla para que puedan crear su primer archivo HTML sin problemas.
Sigue los pasos a continuacion para crear tu archivo HTML y comenzar a desarrollar tus proyectos web.

Pasos para crear un archivo HTML en VS Code

  1. Abre VS Code
  2. Ve a "File" > "New File"
  3. Selecciona Save as y luego como "Nuevo Archivo HTML"
  4. Escribe el código HTML básico
