Kayrox Code

IDE personalizado con soporte Mikota Core

Descripción

Kayrox Code es un entorno de desarrollo integrado (IDE) diseñado para ofrecer una experiencia única y personalizada. Su núcleo está potenciado por Mikota Core, lo que garantiza estabilidad, rendimiento y compatibilidad con proyectos de Kayrox.

Este IDE incluye soporte para el lenguaje Kayrox MikCode (.mkay), resaltado de sintaxis personalizado, temas visuales corporativos y herramientas adaptadas a las necesidades de los desarrolladores.

La visión de Kayrox Code es convertirse en un IDE con identidad propia, integrando funciones modernas y un diseño profesional.

Soporte y compatibilidad

🖥️ Compatibilidad: Windows (Visual Studio 2019, WinForms) y Visual Studio Code mediante extensión personalizada.

🔧 Soporte: Vigente y en desarrollo activo, con actualizaciones periódicas que incluyen mejoras de estabilidad, nuevas funciones y optimización del núcleo Mikota Core.

📦 Estado: Proyecto en evolución, con versiones beta disponibles para pruebas.
