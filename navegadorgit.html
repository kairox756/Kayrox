Navegador Git

Un intento de navegador experimental

Descripción

Navegador Git fue un proyecto experimental que buscaba ofrecer una experiencia de navegación ligera y personalizada, con integración a repositorios y funciones de prueba.

A pesar de las ideas innovadoras, el proyecto nunca alcanzó una versión estable y finalmente se decidió finalizar su soporte.

Nota: Se recomienda explorar otros proyectos vigentes de Kayrox como Flipi.
